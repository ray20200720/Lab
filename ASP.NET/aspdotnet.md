# Web Service
* [如何建立ASP.NET Web Service(.asmx)](http://wordpress.bestdaylong.com/blog/archives/19653)
.asmx是webservice服務程式的字尾名，ASP.NET 使用.asmx 檔案來對Web Services的支援。
.asmx 檔案和.aspx檔案一樣都屬於文字檔案。它包含在.aspx檔案之中，成為ASP.NET應用程式的一部分。
